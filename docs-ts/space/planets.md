## Description

Useful constants from planets and other common celestial objects in our solar system

## Usage

### Earth

```ts
import {
  EARTH_RADIUS,
  EARTH_RADIUS_EQUATORIAL,
  EARTH_RADIUS_POLAR,
  EARTH_CIRCUMFERENCE,
  EARTH_LOWEST_ALTITUDE,
  EARTH_HIGHEST_ALTITUDE
} from 'gis-tools-ts'
```
