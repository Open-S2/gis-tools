# Readers

## Readers I am interested in eventually supporting

- [ ] CSV
- [ ] KML & KMZ
- [ ] XML
- [ ] [flatgeobuf](https://flatgeobuf.org/) & flatbuffers
- [ ] GTFS
- [ ] netCDF (currently only partially supported via nadgrid)
- [ ] gpkg (Geopackage)
- [ ] gml
- [ ] gdb (in gzipped form)
- [ ] TIFFs (.tif, .tiff, .dem )
- [ ] Images (.png, .jpg, .gif)
- [ ] LAS/LAZ
